Proyecto de Página de Películas

Explora nuestro sistema que genera películas basadas en una base de datos, con una arquitectura separada entre backend y frontend.

Acerca del Proyecto

Este proyecto consiste en una página de películas que utiliza un backend y frontend separados para generar y mostrar películas de manera dinámica.

El objetivo principal de este proyecto es proporcionar una plataforma robusta para visualizar películas, donde el contenido es generado en base a datos almacenados en una base de datos.

Backend

El motor que alimenta la generación de películas.

El backend es responsable de gestionar la lógica del servidor y la interacción con la base de datos. Se encarga de procesar las solicitudes del frontend, acceder a los datos almacenados y generar la información necesaria para mostrar las películas.

Frontend

La interfaz visual que los usuarios experimentan.

El frontend es la parte visible de la aplicación, donde los usuarios pueden interactuar con el contenido. Se encarga de presentar las películas generadas por el backend de manera amigable y accesible, utilizando tecnologías como HTML, CSS y JavaScript.

Base de Datos

El corazón del proyecto, donde se almacenan las películas.

La base de datos almacena toda la información relacionada con las películas, incluyendo títulos, descripciones, géneros y más. Es consultada por el backend para generar la lista de películas que luego es enviada al frontend.
